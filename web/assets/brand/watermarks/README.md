# Watermarks

Recommended usage:
- ptmd-watermark-white.png: use on dark footage
- ptmd-watermark-dark.png: use on bright footage
- ptmd-watermark-small.png: use on short-form clips
- ptmd-watermark-transparent.png: use where a subtle overlay is required

Keep watermark opacity between 35% and 60% depending on footage.
Default placement: bottom-right corner with safe margins.
