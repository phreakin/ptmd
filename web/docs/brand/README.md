# Paper Trail MD Brand Assets

This folder is the brand reference and asset index for Paper Trail MD.
It includes the supporting docs that describe brand usage, plus links to the current asset layout under `assets/brand`.

## Brand docs in this folder
- `brand_colors.json`: brand color definitions
- `posting_schedule.json`: posting schedule data
- `social_images.json`: social image metadata and references
- `social_restrictions.json`: platform or usage restrictions
- `social_time_limits.json`: timing limits for social formats
- `teaser_script.md`: teaser copy and script notes
- `web_old_dir.txt`: legacy directory reference

## Current asset areas under `assets/brand`
- `logos/`: master logos and icon assets
- `overlays/`: lower thirds, stamps, and UI textures
- `watermarks/`: watermark variants for video
- `intros/`: social platform intro assets
- `thumbnails/`: thumbnail templates and exports
- `images/`: supporting brand image files and subfolders

## Present files and placeholders
- Brand image files currently live at the top level of `assets/brand` and inside `assets/brand/images`.
- `logos/`, `watermarks/`, `intros/`, and `thumbnails/` exist but may still be mostly placeholder or work-in-progress assets.
- `ptmd_animated_chat_overlay.zip` is a packaged overlay asset.
- `brand-manifest.json`, `brand-guide.md`, and `posting-guide.md` provide additional brand reference material.

## Planned but not documented here
- Video masters, social cutdowns, exports, and audio assets are part of the broader production pipeline, but they are not tracked in this folder.
