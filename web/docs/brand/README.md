# Paper Trail MD Media Asset Structure

This directory contains the base asset layout for Paper Trail MD.

## Key Areas
- assets/brand/logos: master logos and icon assets
- assets/brand/watermarks: watermark variants for video
- assets/brand/overlays: lower thirds, stamps, and UI textures
- assets/brand/intros: social platform intro assets
- assets/brand/thumbnails: thumbnail templates and exports
- video/cases: full documentary masters
- video/clips: cutdowns for social
- video/exports: platform-specific exports
- audio: intro and effects audio

## Notes
Most media files are placeholder files created by script.
Replace them with production-ready PNG, SVG, MP4, MP3, or WAV assets as they are finalized.
