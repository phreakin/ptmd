🎨 Core Brand Palette
🕵️ Primary Colors (Foundation)
5
Midnight Black — #0B0C10
→ Main background (serious, documentary tone)
Paper White — #F5F5F3
→ Text, contrast, “document” feel
Deep Navy — #1C2A39
→ Sections, headers, subtle depth
Slate Gray — #2F3A40
→ Secondary UI, cards, containers
🔍 Accent Colors (Identity + Energy)
6
Evidence Red — #C1121F
→ Alerts, key moments, titles, “this matters”
Highlighter Yellow — #FFD60A
→ Callouts, emphasis (like marking a document)
Forensic Teal — #2EC4B6
→ Data points, links, analytical visuals
Muted Gold — #BFA181
→ Premium feel (used sparingly)
🧠 How It Should Feel
Like a case file on a desk
Like someone uncovered something they shouldn’t have
Clean, but with tension underneath
⚖️ Usage Rules (important)
80% dark (black/navy)
15% neutral (white/gray)
5% accents (red/yellow/teal)

👉 Red and yellow should feel intentional, not decorative.

🔤 Typography Pairing (quick add-on)
Headlines: Bold, condensed
e.g. Bebas Neue, Oswald
Body: Clean, readable
e.g. Inter, Roboto
